React To-Do Application

A simple and interactive To-Do App built using React JS.
Users can add tasks, mark them as complete, and view the list dynamically without page reload.

Features
✔ Add New Tasks

Users can type a task in the input field and click the Add button.

✔ Validation Using Alerts

If input is empty → shows an alert: "Please enter a task!"

On successful add → shows an alert: "Task added successfully!"

✔ Mark as Completed

Each task has a checkbox to toggle the completed state.

✔ Dynamic Rendering

Tasks update instantly using React state without reloading the page.

✔ Clean & Attractive UI

Styled with gradients, spacing, and clean layout.
